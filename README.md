## 簡介

這是 Skilfor 後端原始碼，採用 Express + Sequelize 開發。

## 建置

### 前置作業

複製`.env.example`並改名成`.env`，並且設置裡面的環境變數：

* 資料庫資訊
  * DB_USERNAME
  * DB_PASSWORD
  * DB_DATABASE
  * DB_HOST
* jwt 資訊
  * PW_SALTROUNDS
  * MB_SECRETKEY
* 學生初始點數
  * MB_INITIALPOINTS

完成後 `npm install` 安裝所需套件。

### 開發

1. `npm run dev`

### 部署

1. `npm run pm2`
